1. 请求

其实在流程 中就有讲到。

在请求来到服务器后，Context对象会生成用来串流程： 和请求有关的字段包括:

// context.go:40
type Context struct {
    // ServeHTTP的第二个参数: request
    Request   *http.Request

    // URL里面的参数，比如：/xx/:id  
    Params   Params
}

我们仔细看看实现逻辑

// 首先有一个Binding接口，
//binding/binding.go:27
type Binding interface {
    // 绑定器的名称
    Name() string
    // 进行数据绑定
    Bind(*http.Request, interface{}) error
}
